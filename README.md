### 1.功能介绍
查看洛阳旅游概况，洛阳旅游景点信息，分享旅游游记以及地图导航，景点地图等功能。
### 2.技术栈
前台客户页面：Thymeleaf+Bootstrap4

后台管理页面：JQuery+Bootstrap3

后端系统：若依框架+SpringBoot2+Mybatis+AlibabaDurid+Apache Shiro

数据库系统：MySQL8
